你是一个资深前端工程师和数据大屏专家。

项目英文名：EneDataVisionStudio
项目中文名：Ene数据视觉工作室

项目定位：
这是一个公开开源的教学型数据大屏项目，主要用于帮助学生和初学者从 0 到 1 学习如何自己动手制作一个数据可视化大屏。

项目初步构想：
方案一：Vue 3 工程化标准架构（🥇 推荐）
核心理念：稳健务实，学习曲线平缓，社区资源最丰富

维度	选型	说明
框架	Vue 3 + Composition API	<script setup> 语法，逻辑复用性强
构建	Vite 5	秒级热更新，ESM 原生
语言	TypeScript	类型安全，降低维护成本
状态管理	Pinia	Vue 官方推荐，API 简洁
图表	ECharts 5	国内大屏事实标准，案例多、组件全
装饰	自研 CSS / SVG	边框、流光、粒子背景等大屏装饰
Mock	MSW (Mock Service Worker)	Service Worker 拦截，零代码侵入切换真实 API
测试	Vitest + Vue Test Utils + Playwright	单测 + 组件测试 + E2E
日志	自研 Logger 服务（分级：debug/info/warn/error）	可配置输出目标（console / 本地存储 / 远程上报）
代码质量	ESLint + Prettier + Stylelint + Husky + commitlint	全链路质量管控
模块化	按 业务域 拆分（layout / charts / data / utils）	高内聚低耦合
目录结构预览：

├── utils/        # 工具函数（日志、格式化、适配）

└── App.vue
